1. 什么是动态规划

某一问题有多个重叠子问题。


💡 与贪心不同之处在于：贪心没有状态推导，而是从局部直接选最优的。而动态规划中每个状态一定是由之前的状态推导出来。

1）01背包

一件物品只能放入一次

1. 二维dp数组01背包

1）dp定义：dp[i][j]表示从下标为[0-i]的物品里任意取，放进容量为j的背包里，价值总和最大是多少。 （此时的j表示的下一个物品能放的最大容量）

2）递推公式：

  • 第i个，不放物品：dp[i][j] = dp[i-1][j]
  • 第i个，放物品：dp[i][j] = dp[i-1][j-weight[i]] + value[i]
    所以递归公式：dp[i][j] = max(dp[i-1][j],dp[i-1][j-weight[i] + value[i])

3）初始化：
从递推公式来看，需要初始化 i = 0 ，以及 j = 0 的情况

  • j = 0：无论选什么物品，dp[i][0] = 0，因为不能放入包中。

  • i = 0：表示放入下标为0的物品，dp[0][j] = dp[0][j-weight[0]
    当 j >= weight[0] 时 dp[0][j] = value[0]
    当 j < weight[0]时 dp[0][j] = 0

    其他下标的初始化：
    可以由左上方的数值推测出，可以均赋值为0

2. 一维dp数组（滚动数组）

dp[i][j] = max(dp[i-1][j],dp[i][j-weight[i]] + value[i])
其实可以发现如果把dp[i - 1]那一层拷贝到dp[i]上，表达式完全可以是：dp[i][j] = max(dp[i][j],dp[i][j-weight[i]]+value[i])

这样的话，i之间不存在差别，所以考虑去掉i这个维度，只保留dp[j]

滚动数组：需要满足的条件是上一层可以重复利用，直接拷贝到当前层。

1）dp定义
dp[j] 表示：容量为j的背包，所背的物品价值可以最大为dp[j]。

2）递推公式
dp[j] = max(dp[j],dp[j-weight[i]] + value[i]) 不放物品i，放物品i
相对于二维dp数组的写法，把i维度去掉了。

3）初始化
dp[0] = 0
如果题目给的价值都是正整数，那么非0下标都初始化为0即可

4）遍历顺序

  • 背包倒序遍历
    保证物品i只被放入一次（倒序的时候，前面的状态还是初始状态）
    为什么在二维数组中可以正序遍历：因为在二维数组的推导过程中，前一个的结果不会影响后一个的结果，即不会重复放入。

  • 先遍历物品嵌套遍历背包容量

    如果遍历背包容量放在上一层，那么每个dp[j]就<span style="background:#b1ffff">只会放入一个物品</span>。

对于纯完全背包问题，完全背包中，两个for循环的先后循序，都不影响计算dp[j]所需要的值（这个值就是下标j之前所对应的dp[j]）。

完全背包的应用：

  1. 组合数（1，5）

    • 必须外层for遍历物品，内层for遍历背包。
    • 外层物品遍历，控制了在背包中加入物品的顺序
    • 如果把遍历nums（物品）放在外循环，遍历target的作为内循环的话，举一个例子：计算dp[4]的时候，结果集只有 {1,3} 这样的集合，不会有{3,1}这样的集合，因为nums遍历放在外层，3只能出现在1后面！
    1. 排列数（1，5）（5，1）
    • 外层for遍历背包，内层for遍历物品
    • 背包容量的每一个值，都是经过 1 和 5 的计算，包含了{1, 5} 和 {5, 1}两种情况。

  2. 求最小个数 （顺序都可以）

  • 有顺序和没有顺序都可以，都不影响最小个数。

3）多重背包

有N种物品和一个容量为V 的背包。第i种物品最多有Mi件可用，每件耗费的空间是Ci ，价值是Wi 。求解将哪些物品装入背包可使这些物品的耗费的空间 总和不超过背包容量，且价值总和最大。

每件物品最多有Mi件可用，把Mi件摊开，其实就是一个01背包问题了。

4）总结

递推公式 🌟

  1. 问能否能装满包（或者最多能装多少）：
    dp[j] = max(dp[j] , dp[j-nums[i] + nums[i]])

  2. 问装满背包有几种方法：
    dp[j] = dp[j] + dp[j-nums[i]]

  3. 问背包装满的最大价值：
    dp[j] = max(dp[j] ,dp[j-weight[i]] + value[i])

  4. 问装满背包所有物品的最小个数：
    dp[j] = min(dp[j] , dp[j-coin[i]]+1)

遍历顺序 🌟
  1. 01背包
    二维dp数组01背包先遍历物品还是先遍历背包都是可以的，且第二层for循环是从小到大遍历。
    一维dp数组01背包只能先遍历物品再遍历背包容量，且第二层for循环是从大到小遍历。
  2. 完全背包
    一维dp数组实现，先遍历物品还是先遍历背包都是可以的，且第二层for循环是从小到大遍历。
  • 如果求组合数就是外层for循环遍历物品，内层for遍历背包。
  • 如果求排列数就是外层for遍历背包，内层for循环遍历物品。

打家劫舍问题

1) 如果两间相邻的房屋在同一晚上被小偷闯入，系统会自动报警。

  • dp[i] 表示考虑偷下标为i的房屋，能获得最大的金额
  • dp[i] = max(dp[i-2]+value[i],dp[i-1]);
  • dp[0] = value[0]
  • dp[1] = max(value[0],value[1]);

2) 这个地方所有的房屋都 围成一圈 ，这意味着第一个房屋和最后一个房屋是紧挨着的。同时，相邻的房屋装有相互连通的防盗系统，如果两间相邻的房屋在同一晚上被小偷闯入，系统会自动报警 。

如何考虑成环的情况：
（因为首元素和尾元素不可能同时被偷，所以分成两种情况讨论）

  • 考虑偷首元素，不包含尾元素
  • 考虑偷尾元素，不包含首元素
  • 取以上两种情况的最大值

3）地区只有一个入口，我们称之为“根”。 除了“根”之外，每栋房子有且只有一个“父“房子与之相连。一番侦察之后，聪明的小偷意识到“这个地方的所有房屋的排列类似于一棵二叉树”。 如果两个直接相连的房子在同一天晚上被打劫，房屋将自动报警。

1）判断子序列

给定字符串 s 和 t ，判断 s 是否为 t 的子序列 -> 转化为相同子序列长度是否为t的长度 （这道题也可用双指针和贪心做）

  • dp[i][j]定义：dp[i][j]表示以下标i-1为结尾的字符串s，和以下标j-1为结尾的字符串t，相同子序列的长度。
  • 递推公式：
    在确定递推公式的时候，首先要考虑如下两种操作，整理如下：
    • if (s[i - 1] == t[j - 1])
      • t中找到了一个字符在s中也出现了
      • dp[i][j] = dp[i-1][j-1]+1
    • if (s[i - 1] != t[j - 1])
      • 相当于t要删除元素，继续匹配
      • dp[i][j] = dp[i][j-1]
  • 初始化：
    从递推公式可以看出dp[i][j]都是依赖于dp[i-1][j-1和 dp[i][j-1]，所以dp[0][0]和dp[i][0]是一定要初始化的。

2）不同子序列 🌟

给定一个字符串 s 和一个字符串 t ，计算在 s 的子序列中 t 出现的个数。
（这道题目如果不是子序列，而是要求连续序列的，那就可以考虑用KMP。）

  • dp[i][j]定义：dp[i][j]表示以下标i-1为结尾的字符串s中出现以下标j-1为结尾的字符串的个数。
  • 递推公式：
    • if (s[i - 1] == t[j - 1])
      • 此时dp[i][j]由两部分组成
      • 第一部分是s[i-1] 用来匹配，此时匹配的字符串个数为 dp[i-1][j-1]
      • 第二部分是s[i-1] 不用来匹配，此时匹配的字符串个数为 dp[i-1][j]
    • if (s[i - 1] != t[j - 1])
      • 相当于s要删除元素，继续匹配
      • dp[i][j] = dp[i-1][j]

3）两个字符串的删除操作

给定两个单词 word1 和 word2，找到使得 word1 和 word2 相同所需的最小步数，每步可以删除任意一个字符串中的一个字符。
（与上一题，不同的子序列相比，实际上就是两个字符串都可以删除了）

  • dp[i][j]定义：dp[i][j]表示以下标i-1为结尾的字符串word1，以下标j-1为结尾的字符串word2，如果要相同的话，最少需要几步删除操作。
  • 递推公式：
    • if (word1[i - 1] == word2[j - 1])
      • 此时不需要进行删除操作
      • `dp[i][j] = dp[i-1][j-1]
    • if (word1[i - 1] != word2[j - 1])
      • 此时考虑是删除word1 还是word2 还是同时删除word1 和 word2
      • dp[i][j] = min( dp[i-1][j],dp[i][j-1])+1
  • 初始化：
    从递推公式中，可以看出来，dp[0][j]和 dp[i][0]是一定要初始化的。
    dp[i][0]：word2为空字符串，以i-1为结尾的字符串word1要删除多少个元素，才能和word2相同呢，很明显dp[i][0] = i。dp[0][j]同理。

回文

1）回文子串
给定一个字符串，你的任务是计算这个字符串中有多少个回文子串。

具有不同开始位置或结束位置的子串，即使是由相同的字符组成，也会被视作不同的子串。

  • 输入：”aaa”
  • 输出：6
  • 解释：6个回文子串: “a”, “a”, “a”, “aa”, “aa”, “aaa”

判断一个子字符串（字符串的下表范围[i,j]）是否回文，依赖于，子字符串（下标范围[i + 1, j - 1]）） 是否是回文。
所以为了明确这种递归关系，我们的dp数组是要定义成一位二维dp数组。布尔类型的dp[i][j]：表示区间范围[i,j] （注意是左闭右闭）的子串是否是回文子串。

双指针算法：
首先确定回文串，就是找中心然后向两边扩散看是不是对称的就可以了。

2）最长回文子序列

回文子串是要连续的，回文子序列可不是连续的！ 回文子串，回文子序列都是动态规划经典题目。

  • dp[i][j]定义：dp[i][j]表示以i-j，左闭右闭空间下，最长的回文子序列长度
  • 递推公式：
    在确定递推公式的时候，首先要考虑如下两种操作，整理如下：
    • if (s[i] == s[j])
      • dp[i][j] = dp[i-1][j-1]+2
    • if (s[i] != s[j])
      • dp[i][j] = max(dp[i-1][j],dp[i][j-1])
        如果s[i]与s[j]不相同，说明s[i]和s[j]的同时加入 并不能增加[i,j]区间回文子序列的长度，那么分别加入s[i]、s[j]看看哪一个可以组成最长的回文子序列。那么一定是取最大的。
  • 初始化 dp[i][i] = 1
